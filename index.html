Instructions

  • play a card its value is added to the count
  • You win if you are the only player left
  • You lose if you play a card to make the count greater than 99
  • There are powerful magic cards
    • 4: reverses and doesn't add to the count
    • 9: pass the turn
    • 10: -10 to the count
    • Jack: 99 set the count
      Jacks can be played in succession
